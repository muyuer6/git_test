# 添加远程库

先新建一个仓库，假设命名为git_test

通过命令

```
git remote add origin git@github.com:myname/git_test.git
```

将仓库与本地内容关联

下一步，就可以把本地库的所有内容推送到远程库上：

```
git push -u origin master
```

前提是，仓库为空，没有如readme.md之类的文件
